## Local development üõ†

1. Clone the project, then `cd` into it

```
git clone https://github.com/rwxdan/rwxdan.dev && cd rwxdan.dev
```

2. Install all the dependencies

```
yarn
```

3. Start the development server

```
yarn dev
```
